This project builds an agent that attempts to solve the game Wordle using Bayesian inference. Using a list of all possible five letter words, a prior probability distribution is calculated using letter frequency. The agent updates the posterior probability using Bayes’ rule and the feedback it receives after each guess, and selects the word with the highest posterior probability to be the next guess. When compared to a baseline that randomly selects a word from a list of possible solutions, the model using Bayesian inference on average requires a fewer number of attempts to guess the secret word.
